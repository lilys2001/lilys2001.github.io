Mafia Role Generator

CS Kickstart Final Project

Introduction

Growing up, Mafia was one of my favorite games to play with my friends, and so when we played it again during CS Kickstart, I was reminded of how much I used to enjoy it. Although the rules and mechanics of the game are pretty easy to grasp, I noticed that every time I wanted to play the game, there was not an easy way to secretly and randomly assign roles, so my final project at CS Kickstart was developed as a solution for this problem.

Approach

CS Kickstart was my first introduction to Python, so I knew I had to keep my project viable using only the basic skills I had picked up in the 3 days of programming lessons we had. By the time we started our final projects, we had only learned conditional statements, looping, function definitions, and how to take in user input from the terminal. I started by creating a way to randomly assign roles to players, so I did some research on the random library, and I realized that I could just shuffle a list of roles and then pair it back up with the list of player names. Instead, I focused on the "interface" of the game, utilizing the time library to display each player's role in the terminal and then erasing it before the next pair was shown. I continued to build upon the project by adding in the option for "special roles," ones that make the game more fun, but only work with larger groups, and scaled the number of other roles appropriately.

Challenges, Conclusion, & Future Improvements

Because this was my first experience with Python, this program was very simple. However, it does meet all necessary functionalities for secretly and randomly assigning roles for the game. I was really proud of what I had been able to accomplish with only a few days of Python lessons. Although I do not think I will continue developing this program (as I know there are others like it already in the market), I think it was a great first project for me. In the future, I would like to take the same concept and use it when trying to learn mobile development.
